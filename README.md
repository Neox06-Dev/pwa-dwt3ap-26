# GIT
1. Iniciamos un repositorio de GIT en la terminal: git init
2. Configuramos el usuario 
    -git config user.name "[TU NOMBRE]"
    -git confing user.email "[TU EMAIL]"
3. Agregamos los archivos al repositorio: git add README.md
4.  Hacemos el commit (snapshot): git commit -m "[NOMBRE DEL PROYECTO]"
5. Vinculamos el repositorio remoto desde github a la terminal de vscode: git remote add origin https://github.com/Neox06-Dev/pwa-dwt3ap-26.git
6. Realizamos el push git push origin master
